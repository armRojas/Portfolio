Proyectos

Una vista de los proyectos que he estado realizando y explicación del objetivos del mismo y lo aprendido.

Implementación de una VPN

La VPN la usamos para navegar de forma privada en internet, incluso podemos acceder a contenido adicional de otro país (Netflix en USA cuenta con mayor catálogo). Para implementarlo fue necesario instalar una versión de Linux y configurarla como servidor. Aplicando protocolos de redes y configurar los puertos la VPN se logró con éxito. Adiccionalmente se agregó un pequeño website con una pequeña explicación de una VPN.

Análisis sobre el COVID

El COVID llegó de imprevisto para todo el mundo, mucha gente salió perjudicada, la Inteligencia Artificial fue un factor importante para conocer a la gente más propensa a enfermarse y evitar mayor número de contagios.
Se empleó un pequeño dataset de las personas entubadas y enfermedades con las que contaban, con la finalidad de analizar que persona con enfermedad crónica es más propensa a estar entubada. Datos proporcionados por el INEGI.

Sistema de Información JAVA

Es una aplicación que se encuentra conectada a una base de datos de organización, durante este proyecto fue un ejemplo de una pequeña librería, donde como podemos agregar un nuevo usuario, pedir prestamo de algún libro y devolución.

SEPOMEX

Descargar de los datos proporcionados por SEPOMEX. Exportar nuestros datos a una base de datos y usarla en un programa donde sea necesario el uso de direcciones, la gestión es importante porque de no serlo el programa presentaba retrasos con las consultas.

MS Project

Ms Project es un software que nos ayuda a administrar nuestros tiempos en proyectos dentro de nuestra organización, contamos con un pequeño ejemplo sobre la creación de una empresa y los tiempos aproximados establecidos.

Data Warehouse

Un Data warehouse es un repositorio de todos los datos recogidos por una empresa. Los datos de un almacén de datos deben almacenarse de forma segura, fiables, fáciles de solicitar y de gestionar.
Empleamos un proceso ETL, en el que extraemos de una base de datos y lo trabajamos con un archivo excel, para finalmente crear el almacén de datos y utilizarlo para el análisis.

Detección de Frutas

Lograr una detección de algo material y transformarlo a cómputo es necesario de un intermediaro, en esta ocasión se trabajo con un sensor y un arduino, una vez capturado los datos de al menos veinte a treinta veces se logró con un promedio de rango de los productos. Cuando se leía la fruta con los cálculos empleados nos clasifica con respecto a nuestras lecturas.
